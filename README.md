# 🏠 House Price Prediction Using Machine Learning

## Overview

This project predicts house prices based on various property features using Machine Learning and Flask.

## Features

* House Area Prediction
* Bedrooms Analysis
* Bathrooms Analysis
* Floors Analysis
* Parking Availability
* House Age Consideration
* Urban / Suburban / Rural Location Support
* Modern Glassmorphism UI
* Responsive Design
* Real-time Price Prediction

## Technologies Used

* Python
* Flask
* Pandas
* NumPy
* Scikit-Learn
* HTML5
* CSS3
* JavaScript

## Machine Learning Model

Random Forest Regressor

## Dataset Features

* Area
* Bedrooms
* Bathrooms
* Floors
* Parking
* House Age
* Location Type
* Price
